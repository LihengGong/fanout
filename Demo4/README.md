# Demo for pushpin subscription/publish in Django

## Architecture

```  
               +---------------------+                
               |     Django          |                
      +-------->+------------------+ <----------+     
      |        ||                  | |          |     
      |        ||   Event Stream   | |          |     
      | +-------    (Server)       | --------+  |     
      | |      ||------------------| |       |  |     
+-----|-v--+   ||   Django-Grip    | |       |  |     
|          |   ||                  | |    +--v--|----+
| Client 1 |   ++---------^--------+-+    |          |
|          |              |   |           | Client 2 |
+----------+              |   |           |          |
                          |   v           +----------+
                    +------------+                  
                    |            |                  
                    |   Pushpin  |                  
                    +------------+                  
```
